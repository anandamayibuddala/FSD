Working of Hyperlinks in HTML

The <a> tag defines a hyperlink in HTML. It uses the href attribute to specify the URL of the page the link goes to.

4. Target Attribute Summary

  • _self: Opens link in the current tab/window (default).
  • _blank: Opens link in a new tab/window.
  • _parent: Opens link in the parent frame.
  • _top: Opens link in the full body of the window.
