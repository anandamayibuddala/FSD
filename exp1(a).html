Working of Lists in HTML

1. Ordered List

An ordered list displays items in a numbered sequence.

  1. First item
  2. Second item
  3. Third item

2. Unordered List

An unordered list displays items with bullet points.

  • Apple
  • Banana
  • Cherry

3. Nested Lists

Lists can be nested inside each other to create sub-items.

  • Fruits
    • Apple
    • Banana
    • Cherry
  • Vegetables
    • Carrot
    • Broccoli

4. Ordered List inside an Unordered List

You can also put an ordered list inside an unordered list item.

  • Steps to bake a cake:
    1. Preheat the oven
    2. Mix ingredients
    3. Bake the mixture
  • Enjoy your cake!

5. Definition List

A definition list is used to define terms and their descriptions.

HTML
HyperText Markup Language, used for structuring web pages.
CSS
Cascading Style Sheets, used for styling web pages.
JavaScript
A scripting language used to create dynamic content on web pages.
